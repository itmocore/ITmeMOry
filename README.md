# ITMEMORY — История корпусов ИТМО

Цифровой историко-архитектурный справочник о зданиях Университета ИТМО в Санкт-Петербурге.  
Студенческий исследовательский проект, 2024–2025.

---

## 📁 Структура проекта

```
itmo-history/
├── index.html              # Главная страница
├── buildings/              # Страницы отдельных корпусов
│   ├── kronverksky.html    # Кронверкский пр., 49
│   ├── lomonosova.html     # ул. Ломоносова, 9
│   ├── birzhevaya.html     # Биржевая линия, 14
│   ├── grivtsova.html      # пер. Гривцова, 14
│   └── tchaikovsky.html    # ул. Чайковского, 11
├── css/style.css           # Все стили
├── js/main.js              # Данные + карта + таймлайн + карточки
├── js/building.js          # Данные + логика страниц корпусов
├── images/                 # 📸 СЮДА ДОБАВИТЬ ФОТОГРАФИИ (см. ниже)
└── data/buildings.json     # Резервная копия данных
```

## 📸 Как добавить фотографии

### Какие файлы нужны
Поместите фото в папку `images/` с точно такими именами:

| Файл | Корпус |
|------|--------|
| `kronverksky.jpg` | Кронверкский пр., 49 |
| `lomonosova.jpg`  | ул. Ломоносова, 9 |
| `birzhevaya.jpg`  | Биржевая линия, 14 |
| `grivtsova.jpg`   | пер. Гривцова, 14 |
| `tchaikovsky.jpg` | ул. Чайковского, 11 |

### Рекомендуемые фотографии
- **Кронверкский, 49** — фасад с башней и куполом. Ищите на Wikimedia Commons: "Kronvergsky 49 Saint Petersburg"
- **Ломоносова, 9** — фасад с тремя арочными окнами (бывшая церковь). Поиск: "Lomonosova 9 ITMO"
- **Биржевая, 14** — дом Елисеевых на Васильевском острове. Поиск: "Биржевая линия 14 Елисеевы"
- **Гривцова, 14** — треугольный дом в переулке. Поиск: "Grivtsova 14 Saint Petersburg"
- **Чайковского, 11** — особняк Строгановых. Поиск: "Чайковского 11 Строгановы Петербург"

### Где брать бесплатно (с лицензией CC)
- **Wikimedia Commons**: commons.wikimedia.org
- **Яндекс.Панорамы**: Скриншот фасада в хорошем качестве
- **Собственные фото** — лучший вариант!

### Рекомендуемый размер
- Минимум: 800×500 px
- Оптимально: 1200×800 px
- Формат: JPG (сжатое, до 500 КБ)

CSS уже настроен — как только файл появится в папке `images/`, он автоматически подтянется на карточку и в hero-баннер страницы корпуса.

## ✏️ Как изменить текст

Все тексты хранятся в двух файлах (одинаковые данные):
- `js/main.js` — для главной страницы
- `js/building.js` — для страниц корпусов

Найдите нужный корпус по `id` (например `"kronverksky"`) и отредактируйте поля.

## 🚀 GitHub Pages (5 шагов)

1. Создайте репозиторий: github.com → New repository → Public
2. Загрузите файлы: Add file → Upload files → перетащите всю папку
3. Settings → Pages → Source: main / root → Save
4. Подождите 1–2 минуты (следите в Actions)
5. Сайт: `https://ВАШ_ЛОГИН.github.io/itmo-history/`

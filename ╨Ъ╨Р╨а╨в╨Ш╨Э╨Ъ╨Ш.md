# 🖼 Как добавить фотографии зданий

## Шаг 1 — Найдите фотографии

### Где искать (бесплатно, открытая лицензия):

| Корпус | Что искать на Wikimedia Commons |
|--------|--------------------------------|
| `kronverksky.jpg` | "Кронверкский проспект 49" или "2-й дом городских учреждений СПб" |
| `lomonosova.jpg`  | "Улица Ломоносова 9 Санкт-Петербург" |
| `birzhevaya.jpg`  | "Биржевая линия 14 Васильевский остров" или "Дом Елисеевых Биржевая" |
| `grivtsova.jpg`   | "Переулок Гривцова 14" или "Заёмный банк Петербург" |
| `tchaikovsky.jpg` | "Улица Чайковского 11" или "Особняк Строгановых Петербург" |

**Ссылки для поиска:**
- https://commons.wikimedia.org — введите адрес в поиске
- https://pastvu.com — исторические фото Петербурга по адресам
- Яндекс.Панорамы — сфотографируйте фасад экраном

### Идеальный размер фото: 1200×800 px, формат JPG

---

## Шаг 2 — Положите файлы в папку `images/`

Названия должны быть ТОЧНО такими:
```
images/
  kronverksky.jpg
  lomonosova.jpg
  birzhevaya.jpg
  grivtsova.jpg
  tchaikovsky.jpg
```

---

## Шаг 3 — Загрузите на GitHub

1. Откройте репозиторий на GitHub
2. Зайдите в папку `images/`
3. Нажмите **Add file → Upload files**
4. Перетащите все 5 фото
5. Нажмите **Commit changes**

**Готово!** Фото появятся на карточках главной страницы и на баннерах
страниц корпусов автоматически — никаких изменений в коде не нужно.

---

## Если фото нет — ничего страшного

Пока фото не загружены, на карточках отображается красивый
синий градиент с первой буквой названия корпуса.
